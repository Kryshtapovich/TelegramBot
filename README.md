# Simple currency converter telegram bot
After starting the bot you can see 2 buttons:

![alt text](https://github.com/Krishtapovich/TelegramBot/blob/main/Pictures/1.jpg)

Choosing the first one, you'll see current exchange rates for today

![alt text](https://github.com/Krishtapovich/TelegramBot/blob/main/Pictures/2.jpg)

If you tap the second, bot will send you a message with available currencies. You should choose 2 of them after enter the sum and get result

![alt text](https://github.com/Krishtapovich/TelegramBot/blob/main/Pictures/3.jpg)
